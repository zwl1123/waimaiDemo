

##基本命令
yarn install
yarn serve
yarn build
yarn lint

###项目简介
  这是一个基于vue-cli3搭建的一个项目，后台项目采用别人提供的开源项目接口
  利用的技术栈：vue全家桶(vue+vue-router+vuex),还有一些实现功能必备的库，如swiper，better-scroll

##主要功能
  1.使用swiper库实现轮播
  2.实现登录和注册的功能
  3.利用better-scroll的库实现滑动和实时获取滑动的位置
  4.购物车的功能
  5.利用[].filter实现不同条件的过滤
